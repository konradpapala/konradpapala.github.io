Netmon - Network monitor for in Firefox / Chrome

About

Network monitor - shows requests and responses, including the HTTP headers, of the current page, or globally. It is possible to use it at any time, also after the page has loaded. This is useful when there are random errors in some cases and we just want to see what happened, see a list of the past requests. It also displays request count in current tab.

Installing

Firefox

The extension is listed on addons.mozilla.org

Chrome

The extension is listed on Chrome Web Store
